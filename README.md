# EMS Order Management & Production Planning Dashboard

## Overview

Interactive Excel dashboard developed for production planning and order management analysis in a manufacturing environment.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* KPI Cards

## Key Metrics

* Open Orders
* Closed Orders
* Delivery Achievement %
* On-Time Delivery %
* Revenue Analysis

## Features

* Interactive filtering using slicers
* Dynamic KPI tracking
* Customer-wise analysis
* Industry-wise analysis
* Order status monitoring
